### 模写コーディング ③

### 【制作時間】

- PC 画面 → 約 6.5 時間
- スマホ画面 → 約 1.5 時間

### 【意識したこと】

- コードを書いて、どの様に表示されているのかの確認のスピードを上げる様に意識する。
